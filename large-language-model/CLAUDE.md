# 大语言模型 (Large Language Models)

## 学习信息
- **目标**：从零建立 LLM 知识体系
- **策略**：从"语言模型是什么"出发，沿 基础原理 → 架构 → 训练 → 对齐 → 应用 的主线展开，每个模块建立因果逻辑链
- **结构性洞察**：语言是人类知识的通用载体。学会预测语言 ≈ 被迫学会语言中编码的知识和推理。规模放大了这个效应。

## 逻辑链
- LLM 的底层任务 = 给定上下文，预测下一个 token
- 传统编程：人写规则 → 确定行为；ML：从数据学规则 → 行为从数据涌现
- 语言是知识的通用载体 → 学语言 = 被迫学知识 → 规模放大此效应 → 涌现能力
- 模块1：文本 → BPE分词成token → 模型根据上下文窗口内token算概率分布(softmax) → 采样策略选token → 加入上下文 → 自回归循环直到结束
- Token用子词(BPE)而非字/词：高频片段做单元，最小词表覆盖最多文本
- 概率分布是模型输出，采样策略是独立模块（temperature控制随机性）
- 上下文窗口是硬约束，计算量与长度平方成正比
- 模块2：神经元 = 输入×权重+偏置 → 激活函数 → 输出一个数字
- 同层神经元并行（不同权重提取不同特征），层间全连接
- 多层的意义 = 层层组合：单层只能画直线，多层能组合出任意复杂模式
- token → embedding(查找表，token映射为数字向量) → 多层计算 → 最后softmax输出概率分布
- 训练：文本自带答案（下一个token）→ 前向传播算输出 → 算误差 → 反向传播自动调每个权重 → 重复几十亿次 → 权重从随机变有用
- 权重 = 模型学到的全部"知识"，决定网络对任何输入的行为
- 模块3：Transformer = 注意力机制 + 位置编码
- 注意力：每个token生成Q(在找什么)、K(我是什么)、V(我的信息)，Q和K匹配算相关性，按相关性加权吸收V
- 位置编码：把位置信息加到embedding向量上，让网络区分词序
- 完整前向传播链：文本→分词→embedding(一个向量)→加位置编码→从中生成Q/K/V→注意力计算→重复多层→最后一个位置的输出→softmax→概率分布→采样策略选token
- 训练=前向传播+反向传播(调权重)，推理=仅前向传播(权重固定)
