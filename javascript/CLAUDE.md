# JavaScript

## 学习信息
- **目标**：全栈开发找工作，为后续 TypeScript 打基础
- **策略**：以语言核心机制为主线，围绕"JS 引擎如何执行代码"构建理解，而非背 API。重点放在面试高频考点和实际开发中容易踩坑的机制。
- **结构性洞察**：JS 的一切设计都源于"它生在浏览器里"——单线程、事件驱动、动态类型、解释执行，都是这个出身决定的。理解 JS 不是背语法，而是理解它为什么被设计成这样。
